# MR2525 方案模板库

团队协作用的 **模板录入 · 自动编号 · BOM 报价 · 工作流** 系统。适合约 100 个手工 SketchUp 方案（含面板）的统一管理；前期可本地/局域网共用，并 **导出 CSV 同步到飞书多维表格**。

## 快速启动

```powershell
cd "c:\Users\zl450\Nutstore\1\我的坚果云\Cursor\mr2525-template-catalog"
npm install
npm start
```

浏览器打开：**http://localhost:3847**

默认管理员：`admin` / `admin123456`（首次建库；若已改过密码以实际为准。登录后请在「用户管理」修改。）

> 需要 **Node.js 18+**（推荐 22）。首次运行可能出现 `ExperimentalWarning: SQLite` 提示，可忽略。

**修改后端代码后请重启服务**（若端口占用先释放再启动）：

```powershell
Get-NetTCPConnection -LocalPort 3847 -State Listen -ErrorAction SilentlyContinue |
  ForEach-Object { Stop-Process -Id $_.OwningProcess -Force -ErrorAction SilentlyContinue }
cd "c:\Users\zl450\Nutstore\1\我的坚果云\Cursor\mr2525-template-catalog"
npm start
```

启动成功时应看到 `场景手册 PDF: 20260530-handbook-v2`。前端改动请 **Ctrl+F5**，侧栏 UI 版本见 `public/index.html`。

### 局域网同事访问

同一 WiFi/内网下，同事访问 `http://你的电脑IP:3847`。  
若需长期共用，可将 `data/` 目录放在 Nutstore 同步盘，并在一台常开电脑上运行服务。

## 功能概览

| 模块 | 说明 |
|------|------|
| **新建模板** | 表单填关键信息 → 自动生成 `TPL-场景-序号-文件名` 与 slug |
| **模板列表** | 筛选、搜索、**多标签交集**、序号/封面、表头排序、进入详情协作 |
| **详情页** | 基本信息、BOM、参考价、**版本历史**、**外包深化表单**、状态推进 |
| **单价库** | 型材/六通/五金/板材/供应商/非标；六通·五金可附图片 |
| **模板图册** | 已发布方案合成展示图，下载 PNG / PDF 方案清单 |
| **场景库** | 可配置场景、效果图/渲染图、图上标记关联已发布模板、场景手册 PDF |
| **标签库** | 字云浏览全部标签；详情用 `#` 打标（可新建）；按标签筛模板列表 |
| **导出 CSV** | 导入飞书多维表格（主表 + BOM） |

## 协作流程

```
录入员 上传 skp 登记 → 待清单深化
录入员 补全图/尺寸/报价清单/标签 → 待审核
审核员 核对 checklist → 已发布 → 模板图册 / 飞书同步
```

状态只能按流程推进，避免跳步（审核可退回至「待清单深化」；已下架须重新上架再审核）。

**已发布：** 除 **标签** 外不可改；须 **下架** 后编辑，再审核上架（版本 +0.1；换 skp 模型 +1）。详情状态栏按钮为 **下架**。

**上手指南与流程图：** 侧栏 **流程说明**，或仓库 [docs/WORKFLOW.md](docs/WORKFLOW.md)。**2026-06-04 升级摘要：** [docs/UPGRADE-20260604.md](docs/UPGRADE-20260604.md)。

## 自动编号规则

格式：**`TPL-{场景编号}-{4位顺序号}-{文件名}`**

- **场景编号**：见下表（如展厅 `ZT`）
- **顺序号**：同一场景下全局递增（0001、0002…），与旧版 `TPL-场景-文件名-序号` 共用计数
- **文件名**：来自 skp 文件名（去扩展名、过滤非法字符，最多 40 字）

| 场景 | 代码 | 示例编号（skp：`L型展架.skp`） |
|------|------|--------------------------------|
| 展厅 | ZT | TPL-ZT-0001-L型展架 |
| 零售 | LS | TPL-LS-0001-货架 |
| 活动 | HD | TPL-HD-0001-展台 |
| 办公 | BG | TPL-BG-0001-工位 |
| 会展 | HZ | TPL-HZ-0001-摊位 |
| 宠物 | CW | TPL-CW-0001-跳台 |
| 市集 | SJ | TPL-SJ-0001-单联 |
| 其他 | OT | TPL-OT-0001-方案 |

slug 示例：`zhanting-0001-l型展架`（场景拼音前缀 + 序号 + 文件名）

## 参考价计算（对齐《海智详细清单》）

### MR2525 型材
- 出厂参考价 = `ROUNDUP(长度(in) × 1.778 + 42, 0)`（由单价库公式参数计算；**颜色不影响单价**）
- 报价单价 = 出厂价 × 系数（默认 1）
- 小计 = 数量 × 报价单价
- **颜色**：银色 / 木色等，由管理员在单价库 · 型材 Tab 维护；清单中标注用

### 五金配件
- 从配件目录选取（六通 OL5050/OL2550/OL2525、地脚、连接件、灯带等）
- 小计 = 数量 × 出厂单价

### 板材
- 面积(㎡) = 长(mm) × 宽(mm) / 10⁶，边长由 inch × 25.4 换算
- 按㎡计价：件单价 = 面积 × ㎡单价；固定件价：直接填件单价
- 小计 = 数量 × 件单价
- **同一模板可添加多种板材**

### 单价库（管理员维护）

侧边栏 **单价库** 含：**型材公式与型材颜色**、六通、五金配件、板材、**供应商库**、非标件。同事在模板详情中只能**选取**，不能改价。详见 [单价库微调说明](docs/PRICE-LIB-UPDATES.md)。

| 分类 | 维护要点 |
|------|----------|
| **六通** | 名称、型号、对内/对外价；**可选图片**；序号展示 |
| **五金配件** | 名称、规格、单位、链接、**供应商（必填）**、对内/对外价；**可选图片** |
| **板材** | 材质、颜色（RGB / **透明**）、厚度、计价、**供应商（必填）**、CSV 批量导入 |
| **供应商库** | 从五金/板材汇总供应商名；维护联系人、电话、备注 |
| **非标件** | 类别、项目、对内/对外价、备注；与模板「其他」Tab 同步 |

板材颜色支持 **透明**（`color_hex=transparent`，无 RGB）。五金/板材改价后，已引用模板的报价行自动同步。

### 参考价区间
```
总计 = 型材 + 五金 + 板材 + 其他 + 加工费
参考价下限 = round(总计, 百位)
参考价上限 = round(下限 × 1.15, 百位)
```

可在详情页手动覆盖上下限。

### 详情 · 基本信息栏

| 分组 | 说明 |
|------|------|
| 标识 | 名称、负责人、版本（三列） |
| 尺寸 | 宽 / 深 / 高 mm（三列） |
| 文案 | 一句话卖点、报价口径 |
| 价格 | 参考价上下限；**可定制**（`skin_upgrade_enabled`，chip 复选框） |
| 封面 | 缩略图下拉选择实拍/效果图来源，选中即保存 |
| 历史 | 旧模板若有 `panel_note`，详情只读展示；新模板不再编辑该字段 |

表单已移除：**皮肤选配说明**编辑、**询单表单链接**。详见 [详情基本信息栏说明](docs/DETAIL-BASIC-INFO-UPDATES.md)。
