# MR2525 方案模板库

团队协作用的 **模板录入 · 自动编号 · BOM 报价 · 工作流** 系统。适合约 100 个手工 SketchUp 方案（含面板）的统一管理；前期可本地/局域网共用，并 **导出 CSV 同步到飞书多维表格**。

## 快速启动

```powershell
cd c:\Users\zl450\Nutstore\1\Cursor\mr2525-template-catalog
npm install
npm start
```

浏览器打开：**http://localhost:3847**

> 需要 **Node.js 18+**（推荐 22）。首次运行可能出现 `ExperimentalWarning: SQLite` 提示，可忽略。

### 局域网同事访问

同一 WiFi/内网下，同事访问 `http://你的电脑IP:3847`。  
若需长期共用，可将 `data/` 目录放在 Nutstore 同步盘，并在一台常开电脑上运行服务。

## 功能概览

| 模块 | 说明 |
|------|------|
| **新建模板** | 表单填关键信息 → 自动生成 `TPL-场景-序号` 与 slug |
| **模板列表** | 筛选、搜索、进入详情协作 |
| **详情页** | 尺寸、文件上传（封面/效果图/skp）、BOM、参考价、状态推进 |
| **单价库** | 维护默认物料单价 |
| **对外展示** | 已发布方案，一键复制发给客户的摘要 |
| **导出 CSV** | 导入飞书多维表格（主表 + BOM） |

## 协作流程

```
录入员 提交表单 → 待建模
建模员 上传 skp/图、填尺寸 → 待清单报价
清单员 填 BOM + 加工费 → 待审核
审核员 核对 → 已发布 → 复制对外摘要 / 飞书 Doc 链接
```

状态只能按流程推进，避免跳步（审核可退回建模或报价）。

## 参考价计算（对齐《海智详细清单》）

### MR2525 型材
- 出厂参考价 = `ROUNDUP(长度(in) × 1.778 + 42, 0)`
- 报价单价 = 出厂价 × 系数（默认 1）
- 小计 = 数量 × 报价单价

### 五金配件
- 从配件目录选取（六通 OL5050/OL2550/OL2525、地脚、连接件、灯带等）
- 小计 = 数量 × 出厂单价

### 板材
- 面积(㎡) = 长(mm) × 宽(mm) / 10⁶，边长由 inch × 25.4 换算
- 按㎡计价：件单价 = 面积 × ㎡单价；固定件价：直接填件单价
- 小计 = 数量 × 件单价
- **同一模板可添加多种板材**

### 单价库（管理员维护）

在侧边栏 **单价库** 分三类维护，同事在模板详情中只能**选取**，不能改价：

| 分类 | 维护字段 |
|------|----------|
| **六通** | 名称、型号（OL5050/OL2550/OL2525）、单价 |
| **五金配件** | 名称、规格、单位、单价 |
| **板材** | 材质、颜色、厚度(mm)、计价方式(㎡/件价)、单价 |

板材示例材质：免漆板、海洋板、烤漆木板、烤漆金属板、亚克力板等；颜色如木色、金属砂银、透明等。

### 参考价区间
```
总计 = 型材 + 五金 + 板材 + 其他 + 加工费
参考价下限 = round(总计, 百位)
参考价上限 = round(下限 × 1.15, 百位)
```

可在详情页手动覆盖上下限。

## 数据存储

- 数据库：`data/catalog.db`（SQLite，可备份）
- 附件：`data/uploads/{模板id}/`
- 飞书导入模板：`data/feishu/*.csv`

## 与飞书配合

1. 本地库录入、协作、算价  
2. 侧边栏 **导出主表 CSV / 导出 BOM CSV**  
3. 导入飞书多维表格（见 `docs/FEISHU-SETUP.md`）  
4. 对外仍可用飞书 Doc 详情页；主表 `详情链接` 字段粘贴 Doc URL
