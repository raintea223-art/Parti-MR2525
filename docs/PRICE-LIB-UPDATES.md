### 2.3 板材栏

| 变更 | 说明 |
|------|------|
| 序号列 | 同六通 |
| 对内价默认 | 同六通 |
| 选色弹层 | 点击色块预览 → 弹出面板：**RGB 色轮** + **透明** 选项 |
| 透明色 | 选「透明」时颜色名=`透明`，`color_hex=transparent`（无 RGB）；预览为棋盘格 |
| 颜色名 + 预设 | 色块右侧输入颜色名；下方预设色块网格可点选 |
| 供应商列 | 必填 |
| 表头排序 | 可点击排序：材质 / 颜色 / 厚度 / 对外价 / 供应商 |
| CSV 导入 | **上传 CSV**（主按钮）→ 预览 → 重复行弹窗（覆盖/跳过）→ 写入 |
| CSV 模板 | **下载 CSV 模板**（淡灰次要按钮，少用场景） |
| 删除保护 | 检查 `quote_panels` 引用 |

**选色交互：**

1. 点击左侧色块预览 → 打开选色弹层  
2. **RGB 色轮**：选普通颜色，写入 `color_hex`（如 `#C4A574`）  
3. **透明**：一键设置颜色名与 `color_hex`，适用于亚克力等无法 RGB 表示的透明材质  
4. 点击弹层外区域关闭；下方预设色块（含「透明」）仍可快捷选取  

**列宽（2026-05-28 调整）：**

| 列 | 调整 |
|----|------|
| 材质 / 颜色 / 计价 | 加宽，`nowrap` 避免换行 |
| 对外价 / 对内价 | 缩小（约 64px 输入框） |

**CSV 按钮（2026-05-28 调整）：**

- 顺序：**添加板材** → **上传 CSV** → **下载 CSV 模板**
- 样式：与「添加板材」同为 pill 按钮；上传=主色，下载=淡灰

**CSV 必填列：** 材质、颜色、厚度mm、计价单位、对外价、对内价、供应商  
**可选列：** 颜色HEX(可选)、名称(可选)

模板文件：`data/feishu/板材单价库-导入模板.csv`

**重复判定：** 同 `材质 + 颜色 + 厚度_mm` 视为重复。

## 3. 模板联动

### 3.1 库更新 → 模板同步（已有 + 扩展）

| 条目类型 | 同步字段 |
|----------|----------|
| 六通 / 五金 / 板材 | 对外价、名称、规格、材质等（`propagatePriceItemChange`） |
| 仅对内价 | 不同步报价行快照；算价时从单价库实时读取 |
| 非标件 | 对外价、名称、类别、规格 → `bom_lines` |

### 3.2 库删除 → 模板报错

- **删除前：** 检查引用数，> 0 则 HTTP 409，返回 `{ references: { nuts, hardware, panels } }`
- **模板详情：** 加载时若 `price_item_id` 指向已删条目 → 行标红 + 「单价库已失效」徽章 + 顶部警告条

## 4. 数据模型变更

`price_items` 表新增列（`ensureColumn` 迁移）：

| 列名 | 类型 | 用途 |
|------|------|------|
| `link` | TEXT | 五金采购链接（可选） |
| `supplier` | TEXT | 供应商（五金/板材必填） |
| `color_hex` | TEXT | 板材 RGB 色值（可选） |

`price_items_custom` 无新列；暴露已有 `unit_price_internal`、`note`。

**报价清单 `quote_hardware` 迁移（2026-06-02）：**

- 旧库可能仅有 `model` / `catalog_key`，无 `spec`
- 启动时 `ensureColumn` 增加 `spec`，并将历史 `model` 回填到 `spec`
- 否则详情「五金配件」Tab 添加行会报错（`no column named spec`）

**新表 `profile_colors`（型材颜色）：**

| 列名 | 类型 | 说明 |
|------|------|------|
| `id` | INTEGER | 主键 |
| `name` | TEXT UNIQUE | 颜色名称（如 银色、木色） |
| `active` | INTEGER | 1=启用，0=停用 |
| `sort_order` | INTEGER | 排序 |

`quote_profiles.color`：TEXT，与启用颜色名一致。

**新表 `suppliers`：**

| 列名 | 类型 | 说明 |
|------|------|------|
| `id` | INTEGER | 主键 |
| `name` | TEXT UNIQUE | 供应商名称（与单价条目 `supplier` 文本一致） |
| `contact` | TEXT | 联系人 |
| `phone` | TEXT | 联系电话 |
| `note` | TEXT | 备注 |
| `created_at` / `updated_at` | TEXT | 时间戳 |

## 7. 使用说明

### 板材 CSV 导入流程

1. 单价库 → 板材 Tab → 「下载 CSV 模板」
2. 按模板填写后「上传 CSV」
3. 若有重复规格，逐行弹窗选择 **覆盖** 或 **跳过**
4. 导入完成后列表自动刷新

### 删除条目

若条目已被模板报价清单引用，删除按钮会提示受影响模板编号，需先在模板中更换条目后再删。

### 模板失效提示

打开模板详情时，若引用的单价库条目已被删除，页面顶部显示黄色警告条，对应报价行标记「单价库已失效」。

### 供应商库维护

1. 在五金配件或板材 Tab 填写供应商并保存
2. 打开 **供应商库** Tab，新名称会自动出现（或点击「从单价条目同步」补录历史）
3. 填写联系人、联系电话、备注后失焦即保存

### 板材选色（含透明）

1. 单价库 → 板材 Tab → 点击颜色预览色块  
2. 弹层中选 **RGB 色轮** 或 **透明**  
3. 选透明时颜色名自动为「透明」，无需也无法填 RGB  
4. 也可在下方预设色块或文本框中直接选/输入颜色名
