## 2. 版本号规则（审核发布）

| 场景 | 版本变化 | 写入版本历史 |
|------|----------|--------------|
| 首次审核通过 | 规范为 `v1.0` | `first_publish` |
| 下架后修改再发布 | 次版本 +0.1（如 v1.0→v1.1） | `re_publish_minor` |
| 下架后更换 skp | 主版本 +1（如 v1.1→v2.0） | `re_publish_major` |

- **仅** `audit/approve`（通过并发布）升版；skp 上传时不再单独 bumpMajor。
- 版本历史对比上一版 **快照**，记录基本信息与报价清单 diff。

API：`GET /api/templates/:id/version-logs`

## 3. 外包深化表单

| 操作 | 说明 |
|------|------|
| 导出 JSON / ZIP | 含基本信息、报价清单、标签；ZIP 内附 `catalog.csv`（六通/五金/板材单价库快照） |
| 导出空表 | `?empty=1`，供外包离线填写 |
| 导入预览 | `POST .../deepening-form/preview`，校验不写入 |
| 导入应用 | `POST .../deepening-form/apply`，写入模板（须可编辑状态） |

匹配规则：六通/板材按 `price_item_id`；五金 manual 未匹配 → **其他**（非标库）。

## 4. 单价库 · 六通 / 五金图片

| 项 | 内容 |
|----|------|
| DB | `price_items.image_url` TEXT，默认 `''` |
| 存储 | `data/uploads/price-items/{nut\|hardware}/{id}-{名称}.ext` |
| API | `POST /api/price-items/:id/image`、`DELETE .../image`（管理员） |
| 前端 | 添加表单可选 drop-zone；列表 **图片** 列（缩略图 / 迷你上传 / 清除） |
| 联动 | 改图 **不** propagate；删条目 **清** 图片文件 |
