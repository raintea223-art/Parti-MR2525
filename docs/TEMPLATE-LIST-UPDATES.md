# 模板列表 · 实施说明

> 实施日期：2026-05-28 — 2026-06-01  
> 状态：**已实施**

本文档记录 **模板列表** 页的 UI、筛选与交互变更。

---

## 1. 列结构

| 列 | 说明 |
|----|------|
| **#** | 展示用序号（1、2、3…），随当前排序结果变化 |
| **编号** | `template_code`，点击进入详情 |
| **封面** | 72×72 缩略图（`cover_image`）；无封面显示「无」占位；**点击进入详情** |
| **名称** | 点击进入详情 |
| **场景** | 可排序 |
| **状态** | 可排序 |
| **参考价** | 可排序（按下限 `price_min`） |
| **负责人** | 可排序 |
| **更新** | 最后更新时间 |
| **删除** | 仅管理员 |

---

## 2. 排序规则

可排序列：**场景 · 状态 · 参考价 · 负责人**

| 点击次数 | 行为 |
|----------|------|
| 第 1 次 | 升序 ↑ |
| 第 2 次 | 降序 ↓ |
| 第 3 次 | **取消排序**，恢复默认 |

**默认排序：** 创建时间倒序（与改前一致）

| 字段 | 排序依据 |
|------|----------|
| 场景 | `meta.scenarios` 定义顺序（展厅 → 零售 → …） |
| 状态 | 草稿 → 待清单深化 → 待审核 → 已发布 → 已下架 |
| 参考价 | 数值 `price_min`（无值视为 0） |
| 负责人 | 中文 `localeCompare`；空值靠后 |

同值时以 **创建时间倒序** 作为次要排序。

## 7. 2026-06-02 · 已发布模板只读与版本号

| 规则 | 说明 |
|------|------|
| 状态按钮 | **已发布** 时状态推进区仅显示 **下架**（不再显示「→ 已下架」） |
| 可编辑范围 | **已发布** 仅可改 **标签**；其它字段、图片、报价清单、封面均不可改 |
| 修改流程 | 先 **下架** → 编辑 → **重新上架**（待审核）→ 审核通过 |
| 版本 +0.1 | 非首次 **审核通过并发布** 时自动递增次版本（如 v1.0 → v1.1） |
| 版本 +1 | **更换 skp**（已有模型时再次上传）自动递增主版本（如 v1.1 → v2.0） |
| 版本字段 | 详情中只读，不可手工修改 |

涉及：`src/template-guard.js`、`src/version.js`、`src/server.js`、`public/app.js`。

## 8. 2026-05-31 · 详情状态（已下架重新上架） {#8-2026-05-31--详情状态已下架重新上架}

| 变更 | 说明 |
|------|------|
| 已下架 → 待审核 | 详情「状态推进」显示 **重新上架**；点击后进入 **待审核**，并滚动到「审核发布」区 |
| 发布路径 | 须完成 checklist 后点 **通过并发布**；`PATCH` 不能直接改为 `published` |
| 已发布下架 | 编辑可点 **下架**（见 §7；不再显示「→ 已下架」） |
| 待审核 | 状态推进区不显示「→ 已发布」（避免与审核区重复） |

涉及：`src/constants.js`（`archived → pending_review`）、`src/server.js`、`public/app.js`（`requestRelistTemplate`）。

## 10. 2026-06-01 · 列表按标签筛选（交集）

| 能力 | 说明 |
|------|------|
| 筛选 UI | 工具栏卡片内第二行：已选标签 pill、`#添加筛选` 输入、**清除标签** |
| 多标签逻辑 | 同时选中 A、B 时，仅显示**同时带有 A 和 B** 的模板（交集 / AND） |
| 追加筛选 | 从 **标签库** 字云点击标签 → 跳转列表并**追加**该标签（非替换） |
| API | `GET /api/templates?tag=可拆装&tag=宠物` 或 `?tags=可拆装,宠物` |
| 空结果 | 提示「同时包含标签 #A、#B 的模板暂无」 |
| 下拉定位 | 列表专用：`#list-tag-filter-dropdown` 打开时挂 `document.body`，`fixed` 贴输入框，避免被表格遮挡 |

**不再显示** 列表页底部 `formula-hint` 说明段落（能力以本文档为准）。

涉及：`src/db.js`、`src/tags.js`、`public/app.js`、`public/styles.css`。
